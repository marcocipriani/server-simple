# :floppy_disk: server-simple
Testing for a C server app development.

## Installation
1. Clone repo
2. Compile with makefile included
3. Run ```./server [<path>]``` (Default: ~)
3. Run ```./client [<operation-number>]```

### Operation numbers
- **1** list available files
- **2** get a file
- **3** put a file
- **0** exit
